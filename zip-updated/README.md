# ZIP-UPDATED - v25.0.0-rc14 Duzenlenmis Dosyalar

Bu klasorde yeni tasarim ile duzeltilen/güncellenen dosyalar bulunuyor.

## Icerik

### Views (PHP Dosyalari)
- `shared/` - Paylasilan layout ve header dosyalari
- `partials/` - Header, footer partials
- `admin/partials/` - Admin panel partials
- `auth/partials/` - Auth sayfalari partials
- `customer/partials/` - Musteri paneli partials
- `layouts/` - Shell layout dosyalari
- `home/` - Ana sayfa
- `products/` - Urun sayfalari (hosting, vps, ssl, vb.)

### Assets (CSS/JS)
- `ao-full-ui-reset.css` - Ana CSS dosyasi
- `admin/` - Admin CSS dosyalari
- `themes/` - Tema dosyalari
